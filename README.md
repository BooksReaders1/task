# 任务管理系统

一个基于浏览器 IndexedDB 的轻量级任务管理 Web 应用，专为追踪和管理作品任务设计。支持任务增删改查、状态分类、收藏功能、搜索筛选以及统计面板，集成 JM 配置和预览图功能。

## ✨ 特性

### 核心功能
- **任务管理**：创建、编辑、删除任务，支持批量添加
- **状态分类**：三种状态 - 未使用、已完成、已失效
- **收藏功能**：标记喜欢的任务，快速筛选
- **搜索筛选**：实时搜索任务和备注内容
- **统计面板**：直观显示各类任务数量统计
- **数据持久化**：使用 IndexedDB 本地存储，无需服务器

### 高级功能
- **智能跳转**：根据作品 ID 位数自动选择跳转目标
  - 5-8 位：跳转到 JM 作品页
  - 9 位及以上：跳转到 P 站并复制作品 ID
- **预览图**：自动提取任务中的作品 ID 生成预览图
  - JM 类型 : 显示第一张图片
  - Pixiv 类型 : 显示第一张图片
- **本地预览**：为 JM 和 Pixiv 类型提供本地预览功能
  - JM 类型 : 跳转到 `https://booksreaders1.github.io/TestPicture/?id=XXX`
  - Pixiv 类型 : 跳转到 `https://booksreaders1.github.io/pictureForP/?pid=XXX`
- **随机作品**：一键从 Pixiv、JM 类型中随机抽取一个作品并跳转
- **剪贴板粘贴**：一键从剪贴板读取并创建任务
- **数据导出/导入**：支持将全部任务数据导出为 JSON 文件用于备份迁移；提供两种导出方式：「导出数据 」同时复制并下载文件，「导出数据 」仅复制到剪贴板不生成文件；导入时仅允许在空数据库中操作，支持文件导入和粘贴 JSON 文本两种方式
- **侧边菜单**：点击左上角三横线按钮展开黑色小菜单，集中管理配置和导入导出功能
- **响应式设计**：完美适配桌面端和移动端

## 🚀 快速开始

### 使用方法
1. 直接在浏览器中打开 `index.html` 文件
2. 无需安装任何依赖，无需服务器
3. 数据自动保存在浏览器本地

### 创建任务
1. 在输入框中输入任务文本
2. 填写备注信息
3. 点击「保存任务」按钮
4. 或使用「粘贴」按钮从剪贴板快速导入

### 配置 JM
> ⚠️ 注意：配置 JM、恢复默认 JM、恢复默认预览功能暂时屏蔽，点击将提示"功能暂时屏蔽！"

1. 点击左上角「☰」菜单按钮展开侧边栏
2. 点击「⚙️ 配置 JM」或「🔄 恢复默认 JM/预览」
3. 选择配置类型：
   - **JM 跳转地址**：设置 JM 网站的基础 URL
   - **预览图 API**：设置预览图服务的 API 地址
4. 输入新的 URL 并确认

## 🔧 技术细节

### 数据存储
- 使用 **IndexedDB** 进行本地数据持久化
- 数据库名称：`taskDB`
- 数据表：
  - `tasks`：存储任务信息
  - `configs`：存储全局配置

### 作品 ID 提取
- 优先匹配连续的 9 位数字
- 若无 9 位数字，则提取所有数字
- 支持从任务文本中提取多个作品 ID 批量创建任务

### 预览图生成
- 自动从任务文本中提取作品 ID
- 根据配置的预览图 API 生成图片 URL
- 格式：`{API_BASE}{作品_ID}.jpg?u={时间戳}`
- 首次加载失败时可手动刷新

### 跳转逻辑
```javascript

} else if  {
    // 跳转到 P 站主页，并复制作品 ID 到剪贴板
}
```

## 🛠️ 开发说明

### 项目结构
```
/workspace
├── index.html      # 主页面
└── README.md       # 项目文档
```

### 代码特点
- **零依赖**：纯 HTML/CSS/JavaScript 实现
- **单文件**：所有代码在一个 HTML 文件中
- **响应式**：支持移动设备和桌面端
- **异步处理**：使用 Promise 处理 IndexedDB 操作

### 自定义配置
默认配置：
- JM 跳转地址：`https://jm18c-uoi.net/album/`
- 预览图 API：`https://cdn-msp.jm18c-uoi.net/media/albums/`

可通过界面配置按钮修改，配置保存在 IndexedDB 中。

## 📝 使用场景

1. **作品追踪**：记录已购买/想购买的作品
2. **进度管理**：标记作品的阅读/下载状态
3. **快速访问**：一键跳转到作品页面
4. **收藏整理**：管理喜欢的作品列表

## ⚠️ 注意事项

- 数据仅保存在本地浏览器，清除浏览器数据会导致数据丢失
- 建议定期备份重要任务信息
- 预览图加载依赖外部 API，需确保网络畅通
- 弹出窗口被拦截时，链接会自动复制到剪贴板
